Building modern apps with CBWIRE and AlpineJS

In this hands-on, code-driven workshop, Grant Copley and Esme Acevedo will walk you through building a modern web application from scratch using CFML and the powerful ColdBox module CBWIRE. This workshop focuses on blending CBWIRE’s HTML-over-the-wire technology, AlpineJS, and showcases how using the module can greatly enhance your productivity as a developer and simplify your web applications.

Grant Copley
Esme Acevedo
Development and Hosting using Docker, CI/CD, and AWS ECS

This workshop will provide you with live and hands-on examples of developing a web application and hosting it in a containerized environment on AWS ECS and using CI/CD. We will build a basic application, use Docker for our local development environment, use GIT with GitHub for version control, GitHub actions for CI/CD, and AWS ECS to host the website.

Daniel Garcia
Jacob Beers
Getting Started with BoxLang!

Dive into the fundamentals of BoxLang runtimes in this hands-on workshop. Learn how to set up, configure, and optimize BoxLang runtimes for various use cases, from CLI tools to server-based applications.

Brad Wood
Jon Clausen
Luis Majano
"Just Enough” Workshop: Commandbox, Coldbox, CFConfig, Migrations, Quick, Qb, CBValidation

This workshop will guide the attendee through some of the most common elements of creating an API using Box products including CommandBox, Coldbox, CFConfig, CBValidation, migrations, Quick, and QB

Dan Card
Javier Quintero
